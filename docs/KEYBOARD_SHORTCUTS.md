# Keyboard Shortcuts Guide

This document describes all available keyboard shortcuts in the Portal application.

## Command Palette

### Toggle Command Palette
- **macOS**: `Cmd + K`
- **Windows/Linux**: `Ctrl + K`

Open the command palette to search for pages and quick actions. Type to filter and use arrow keys to navigate.

---

## Quick Navigation

### Dashboard/Home
- **macOS**: `Cmd + Shift + H`
- **Windows/Linux**: `Ctrl + Shift + H`

Navigate to the main dashboard.

### Members List
- **macOS**: `Cmd + Shift + U`
- **Windows/Linux**: `Ctrl + Shift + U`

View all members.

### Donations List
- **macOS**: `Cmd + Shift + B`
- **Windows/Linux**: `Ctrl + Shift + B`

View all donations.

### Social Aid Applications
- **macOS**: `Cmd + Shift + A`
- **Windows/Linux**: `Ctrl + Shift + A`

View social aid applications and requests.

---

## Quick Actions

### Create New Member
- **macOS**: `Cmd + Shift + M`
- **Windows/Linux**: `Ctrl + Shift + M`

Open the new member creation form.

### Create New Donation
- **macOS**: `Cmd + Shift + D`
- **Windows/Linux**: `Ctrl + Shift + D`

Open the new donation entry form.

## Command Palette Features

### Recent Pages
View the last 5 visited pages for quick access.

### Search
Type to search through:
- All available pages
- Dashboard sections
- Member management
- Donation tracking
- Social aid applications
- Settings

### Keyboard Navigation in Command Palette
- **Arrow Up/Down**: Navigate through results
- **Enter**: Select highlighted item
- **Escape**: Close command palette
- **Type**: Filter results

## Browser Keyboard Shortcuts

These are standard browser shortcuts that work within Portal:

### Navigation
- **Alt + Left Arrow**: Go back (browser history)
- **Alt + Right Arrow**: Go forward (browser history)
- **F5 or Ctrl + R**: Refresh page

### Page Search
- **Cmd + F (macOS)** or **Ctrl + F (Windows/Linux)**: Find text on current page

### View Source
- **Cmd + Option + U (macOS)** or **Ctrl + Shift + U (Windows/Linux)**: View page source

## Tips for Efficient Navigation

### 1. Use Command Palette for Discovery
If you're unsure about a feature location, open the command palette and search for it.

### 2. Memorize Frequent Shortcuts
The most common operations are:
- `Cmd + Shift + M` - New member
- `Cmd + Shift + D` - New donation
- `Cmd + Shift + U` - Members list
- `Cmd + K` - Open command palette

### 3. Combine with Command Palette
For less frequent operations, use the command palette:
1. Open with `Cmd + K` (or `Ctrl + K`)
2. Type what you're looking for
3. Press Enter

### 4. Recent Pages
The command palette shows your 5 most recently visited pages, making quick navigation easier.

## Accessibility

### Screen Readers
All keyboard shortcuts are accessible to users with screen readers. The command palette provides:
- Proper ARIA labels
- Keyboard navigation indicators
- Status messages

### Keyboard-Only Navigation
Portal supports full keyboard navigation:
- Navigate all menu items
- Activate buttons and links
- Fill out forms
- No mouse required

### Focus Management
- Clear focus indicators
- Logical tab order
- Focus trap in command palette when open

## Troubleshooting

### Shortcut Not Working?

1. **Check your browser**
   - Some shortcuts may conflict with browser shortcuts
   - Try in a different browser

2. **Check for input focus**
   - Shortcuts don't work inside text inputs
   - Click outside the input field first

3. **Check caps lock**
   - Shortcuts are case-sensitive for some keys
   - Make sure caps lock is off

4. **Report issues**
   - If a shortcut consistently doesn't work, contact support

### Shortcut Conflicts

If a keyboard shortcut conflicts with your OS or browser shortcuts:

1. **macOS**: Check System Preferences > Keyboard > Shortcuts
2. **Windows**: Check Settings > Ease of Access > Keyboard
3. **Browser**: Check your browser's extension settings

Most conflicts can be resolved by adjusting browser or OS settings.

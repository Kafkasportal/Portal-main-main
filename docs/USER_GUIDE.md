## ‚ùì Frequently Asked Questions

**Answer**: Yes, if you have regular backups. Restore from latest backup. However, deletion is permanent, so be careful.

**Answer**: Yes, system supports multiple users. Each user logs in with their own account.

- Role-based access control
- Automatic backup system
- Sentry error tracking

### Can I use from mobile devices?

**Answer**: Yes, system has responsive design. You can use it from tablet and phone.

### How do I print reports?

**Answer**: On report page, click "Yazdƒ±r" (Print) button or use Ctrl+P (Windows) / Cmd+P (Mac).
